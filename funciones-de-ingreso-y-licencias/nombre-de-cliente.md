# Nombre de Cliente

Es el nombre de la Empresa o Persona que contrató el Software para que usted como usuario pueda utilizarlo.

El campo de Nombre de Cliente debe ser ingresado justo al lado de la etiqueta **(Cliente:)** en la Ventana de Licencias dentro del Software:

Una ves ingresado el nombre del Cliente, hacer click en el Botón **(Cambiar)** y si el valor es correcto, el campo Rojo cambia a Color Verde.
