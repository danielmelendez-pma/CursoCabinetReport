# Creación de Módulos

<figure><img src=".gitbook/assets/image (27).png" alt=""><figcaption></figcaption></figure>

La Ventaja de Creador de Módulos consta de las siguientes partes:

{% columns %}
{% column %}
<div data-with-frame="true"><figure><img src=".gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure></div>
{% endcolumn %}

{% column %}
Debe seleccionar o crear el Catálogo, Categoría, Subcategoría, Nombre y Código del Módulo a crear o editar.
{% endcolumn %}
{% endcolumns %}

{% columns %}
{% column %}
Debe colocar los valores de medidas preliminares del módulo que va a crear o editar.
{% endcolumn %}

{% column %}
<div data-with-frame="true"><figure><img src=".gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure></div>
{% endcolumn %}
{% endcolumns %}

{% columns %}
{% column width="50%" %}
<div data-with-frame="true"><figure><img src=".gitbook/assets/image (30).png" alt=""><figcaption></figcaption></figure></div>
{% endcolumn %}

{% column width="50%" %}
Revisar y ajustar si se requiere, los parámetros del catálogo seleccionado o creado, esto para que al crear cada pieza con sus fórmulas, se vayan estableciendo las medidas importantes para el cálculo de cada pieza de su modulo que está creando o editando.
{% endcolumn %}
{% endcolumns %}

{% columns %}
{% column %}
En esta sección se crean y editan las fórmulas de cálculo de cada una de sus piezas que se necesiten para su catálogo de módulos, para poder con cada una de estas piezas armar su mueble y poder sea utilizado en sus producciones.
{% endcolumn %}

{% column %}
<div data-with-frame="true"><figure><img src=".gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure></div>
{% endcolumn %}
{% endcolumns %}

{% columns %}
{% column %}
<div data-with-frame="true"><figure><img src=".gitbook/assets/image (33).png" alt=""><figcaption></figcaption></figure></div>
{% endcolumn %}

{% column %}
Sección en la que se asigna una imagen a cada uno de los muebles creados, para que tenga una representación visual de cada mueble.

Esta imagen debe ser creada previamente para poder subirla a su catálogo.

Puede agregarla en fondo Negro o Blanco.
{% endcolumn %}
{% endcolumns %}

En la parte inferior puede colocar la descripción textual del mueble, acceso para agregar herrajes como Accesorios, Ferretería de Montaje y Ferretería de instalación, de esta manera tener un catálogo bien completo.

<div data-with-frame="true"><figure><img src=".gitbook/assets/image (34).png" alt=""><figcaption></figcaption></figure></div>

También puede utilizar las piezas ya creadas en la base de datos pública de Cabinet Report, abriendolas desde el menú lateral, seccion **Creador de Piezas**, se le acoplará en la parte izquierda de la ventana de **Creador de Módulos** para solo seleccionar piezas para armar su mueble:
